1.скопировать ssh значение из нужного репозитория, затем в гитбаш или любом аналогичном терминале вводим команду: git clone "ссылкаssh".
2.Всё также через терминал при помощи команды : git init создаём локальный репозиторий в выбранной папке.
 Затем добавляем во временное хранилище выбранный проэкт командой : git add "путь к папке или файлу который дабавляем"(git add . добавит все файлы в директории в которой лежит файл хранилища)
 После чего коммитим изменения командой git commit -m ""
 прикрепляем локальный репозиторий к удалённому командой git remote add origin main
 И наконец отправляем коммит в удалённый репозиторий командой git push (git push -u origin main если делаем пуш впервые)
3.Запросить изменения можно командой git pull
4.git status показывает информацию о пользователе (почта и имя пользователя)
  git log показывает историю всех коммитов и действий
  git remote -v показывате к какой ветке подключён локальный репозиторий
